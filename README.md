Meet other music lovers on social media network.

- <b>News Feed:</b> Share, comment or like photos, music or music events with your followers or followees on this page.

## Skill
- <b>Language:</b> Kotlin
- <b>User Interface:</b> Activities, Fragment, Lifecycles, ConstraintLayout, RecyclerView, Dialog, DatePicker, Floating Action Button, Media Player
- <b>Jetpack:</b> LiveData, Lifecycles, ViewModel, Navigation, Data Binding
- <b>Patterns:</b> MVVM, Obsever, Singleton
- <b>Firebase:</b> Authentication, Firestore, Storage, Crashlytics
- <b>Animation</b>
- <b>Facebook SDK</b>
- <b>Test:</b> JUnit, Mockito, Espresso
